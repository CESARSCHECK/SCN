Proxy_FEH_Framework
=====================================

This repository contains the source code for the development objects listed in the following SCN blog

http://scn.sap.com/community/pi-and-soa-middleware/blog/2014/04/16/custom-proxy-framework-with-fehech


1) SI_IB_A_PurchaseOrderCreate.wsdl
- WSDL for ESR service interface (Inbound Asynchronous)

2) ZESYII_SI_IB_A_PURCHASE_ORDER~SI_IB_A_PURCHASE_ORDER_CREATE.txt
- Logic for server proxy method of above service interface

3) NUGG_PROXY_FEH_FRAMEWORK.nugg
- SAPLink nugget file containing ABAP classes for Proxy/FEH framework
